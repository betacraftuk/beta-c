## Building

BetaC requires [CMake](https://cmake.org/), [SDL2](https://github.com/libsdl-org/SDL/releases/), and [SDL_image](https://github.com/libsdl-org/SDL_image/releases).

1. `Copy game resources to resources/`
2. `mkdir build`
3. `cd build`
4. `cmake ..`
5. `make`
